
# Nostos - Project Specification

## Overview
Nostos is a travel experience sharing and tracking app that allows friends and family members to share, plan, and track travel experiences together. Built as a hobby project to explore AI development and have fun.

## Purpose
Enable users to collaboratively view and track travel experiences—both planned and completed—in a shared, visual format.

## MVP Features

### Core Functionality
- **Experience Grid Display**: Visual grid layout showing travel experiences
- **Hard-coded Experience Data**: Initial MVP uses static travel experience list
- **Experience Cards**: Each card displays:
    - Image
    - Title

### User Experience
- Clean, intuitive grid interface
- Quick visual scanning of travel experiences
- Grouping by friends/family members (future enhancement)
- Responsive design
